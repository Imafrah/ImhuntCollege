## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Environment Variables

| Name | Value |
| --- | --- |
| `NEXT_PUBLIC_API_URL` | `https://imhuntcollege.onrender.com` |

Copy `.env.local.example` to `.env.local` for local development.

## Deploy to Vercel

1. Import the `frontend` project in Vercel.
2. Set `NEXT_PUBLIC_API_URL` in the Vercel dashboard.
3. Deploy with the default Next.js build.

## API

Backend API: `https://imhuntcollege.onrender.com`
